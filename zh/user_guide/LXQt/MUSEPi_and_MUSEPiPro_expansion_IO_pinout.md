# MUSE Pi/MUSE Pi Pro扩展IO定义

MUSE Pi/MUSE Pi Pro 主板上都有外扩IO，方便用户扩展外设，比如GPIO设备，I2C设备，SPI设备，在**Bianbu Star V2.1.5**及以后的版本，对这些扩展IO功能进行了初始化默认定义，用户可以不用去重新修改dts。
